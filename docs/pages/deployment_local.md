# Local Development Setup

This guide covers setting up Safex for local development.

## Prerequisites

### Required Tools

- **Go**: 1.21 or later
- **Rust**: 1.70 or later with `wasm-pack`
- **Node.js**: 18 or later with npm
- **Make**: GNU Make for build automation
- **Git**: For version control

### Build Dependencies

```bash
make build
```

### Run locally

```bash
make run
```
